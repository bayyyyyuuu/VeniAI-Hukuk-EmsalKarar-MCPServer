# Changelog

All notable changes to this project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-21

### Added
- Initial project release.
- Yargitay precedent search engine (stdio-based MCP).
- 3-tier caching architecture (Memory, PostgreSQL, Browserless).
- Bun runtime integration.
- Docker and Docker Compose configuration.
- GitHub Actions CI workflow.
- High-standard repository boilerplate (EditorConfig, Husky, Lint-staged, Commitlint).
- Comprehensive environment variables template.
- Security policy and contribution guidelines.
